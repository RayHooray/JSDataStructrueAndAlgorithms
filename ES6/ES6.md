# 基本数据类型和引用数据类型
### 基本数据类型
```
基本数据类型包括(5 + 1)， string, number, boolean, undefined, null, symbol。
基本数据类型是按值访问的，所以可以操作保存在变量中的实际值

引用数据类型包括：Object、 Function、Set、Map。。。。。
Function 本质上也是一种 Object 类型，但是，由于具有一些特殊的属性，所以有必要将 Function 和 Object 进行一个区分

引用数据类型的值是保存在内存中的，JS 不允许直接访问内存中的位置，不能直接操作对象的内存空间，所以在操作对象的时候，其实是在操作对象的引用，而非实际对象。

引用数据类型的值是按引用访问的
```
